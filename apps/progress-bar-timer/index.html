Progress Bar Timer

Progress Bar Timer

画面全体のプログレスバーで時間を管理。

画面全体のプログレスバーで進捗を一目で把握。横向き専用、Live Activity 対応のシンプルなタイマー。

横向き専用 Live Activity iPhone iOS 17+
Download on theApp Store

使い方を、ひと目で。

設定して、再生して、見守るだけ。

モード選択 → 時間設定 → 再生

砂時計でカウントダウン、ストップウォッチでカウントアップ。3 タップで開始。

画面全体がプログレスバー

離れた場所からも進捗が一目でわかる、視認性最優先のモノクロデザイン。

ライト / ダーク両対応

システム連動・ライト・ダークから選択。

ロック画面 / Dynamic Island

Live Activity で残り時間とバーを常時表示。再生・一時停止・リスタートもアプリを開かずに。

2 つのモード。

時間の見方は、向きで選ぶ。

⌛

カウントダウン

設定した時間からゼロへ。残り時間がバーで埋まっていく。集中作業のポモドーロや料理の時間管理に。

⏱

カウントアップ

ゼロから設定した時間まで計測。経過時間がバーで進んでいく。運動やプレゼンのリハーサルに。

どちらのモードも、設定時間に到達すると自動停止して
選択したアラーム音(Beep / Chime / Bell / Digital)で通知。

すべてが、集中のために。

余計な要素を削ぎ落とした、最低限の機能セット。

Live Activity

ロック画面と Dynamic Island に進捗バーと残り時間を常時表示。再生 / 一時停止 / リスタートをアプリ外で操作。

4 種類のアラーム

Beep / Chime / Bell / Digital + なし。システム音量と独立したアプリ内音量スライダーも搭載。

最大 24 時間

hr / min / sec のホイールで自由に時間を設定。秒単位の短時間から、終日タイマーまで。

バックグラウンド対応

ホームに戻っても、画面をロックしても、設定時間が来ればローカル通知で知らせる。

テーマ切替

システム連動 / ライト / ダークの 3 種類。完全モノクロのデザインで、どのテーマでも視認性が高い。

完全オフライン

ネットワーク通信ゼロ。広告もトラッキングもなし。すべてが端末内で完結する。

こんな方におすすめ

  • ポモドーロやタイムブロッキングで集中作業をしたい方
  • 運動やストレッチの時間を計りたい方
  • プレゼンテーションや面接のリハーサル
  • 料理の調理時間を計りたい方
  • 横向き画面で離れた場所からも進捗を確認したい方
  • シンプルで広告のないタイマーを探している方
