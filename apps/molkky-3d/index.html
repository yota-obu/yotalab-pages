Mölkky 3D

3D Physics Mölkky

指 1 本で、ピンが転がる。

フィンランド発祥のアウトドアスポーツ「モルック」を、本格 3D 物理シミュレーションで再現。スワイプで投げて、ピンが豪快に倒れる爽快感を片手で味わえます。

3D 物理 マルチプレイ CPU 戦 iPhone iOS 14+

ルールはシンプル、
戦略は奥深い。

12 本のピンを倒して、ちょうど 50 点で勝利。

1 本だけ倒れたらピンの番号、2 本以上ならその本数が得点。
50 点を超えると 25 点に戻るので、終盤は数字選びが勝負。

ひとりでも、みんなでも。

気分に合わせて、ふたつの遊び方。

👥

みんなで対戦

1 台の iPhone を回して、2〜4 人で交代に投げるローカルマルチプレイ。集まったその場で、すぐに始められます。プレイヤー名のカスタマイズも可能。

🤖

CPU と対戦

3 段階の難易度から選べる CPU 戦。初級はおだやかに、中級はそこそこ正確、上級はノイズゼロの完璧投擲で挑んでくる強敵です。

気持ちよさのために、
すべて。

スワイプ操作と物理シミュレーションを、徹底的にチューニング。

🔥

本格 3D 物理

Jolt Physics エンジンで、棒の回転・ピンの倒れ方・連鎖まで全てリアルに再現。同じ投げ方でも結果が変わる本物の感触。

👆

スワイプで投擲

下から上へのスワイプの長さと速さで、初速・打ち出し角を直感的に決定。チュートリアルなしで誰でもすぐに投げられます。

🎯

縦・横の使い分け

棒タップで横持ち / 縦持ちを切替。ジャベリン投げで遠くの単独ピンを狙撃するか、横持ちで密集に大ダメージを与えるか。

📊

リアルタイム計器

画面下部に ANGLE / DRIFT / POWER の 3 つのメーター。投擲中の挙動を可視化し、上達がそのまま指先に伝わります。

🔅

ローカル完結

通信ゼロ・課金ゼロ。電波が届かない場所でも、いつでも誰とでも遊べます。アカウント登録も不要。

🎤

日本語 / 英語

アプリ内設定からいつでも切替。BGM / 効果音の ON/OFF も自由にカスタマイズできます。
